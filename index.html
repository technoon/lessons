Lession -1 : Node.js, Express, EJS & MongoDB

In this lesson, let see how to do the CRUD operations on MongoDB using Node.js, Express & EJS. I am assuming you are familiar with Node.js & MongoDB. Express is a web application framework for Node.js, which provides the easy way to build web applications with multi pages. Express comes with 'Jade', a templating engine. EJS is also a popular templating engine for Node.js.

How to use Express & EJS

Install the 'express' & 'ejs' to the node_modules using the below command.

npm install express
npm install ejs

Let's how to use the express for handling the http requests. use the require method to import the express library to the node script. From the below code the express object 'app' is passed as an argument to the 'createServer' method, after this all the incoming http get calls will be handled by the express framework. Save the below script as 'mongocrud.js' and run the script as 'node mongocrud.js'. Access the localhost:3000 in the browser window, Bingo ! Welcome to Technoons ! 


var express = require('express'),
http = require('http');

var app = express();

app.get('/', function(req, res) {
res.send('Welcome to Technoons!');
});

http.createServer(app).listen(3000, function() {
console.log("Server is listening on port 3000"); 
});

Configure the express by setting the essential parameters. Take a note that 'ejs' is mentioned as the view engine


app.set('views', __dirname + '/views');
app.set('view engine', 'ejs');
app.use(express.bodyParser());
app.use(express.methodOverride());
app.use(app.router);
app.use(express.static(path.join(__dirname, 'public')));

Now, let's see how to display html elements using EJS. In the mongocrud.js add the below code fragment. So when the request comes the contents in the index.js will be rendered in the browser. Post this create a file index.ejs and include the below html fragments. 


app.get('/', function(req,res) {
res.render('index.ejs');
});

Run the mongocrud.js and access the 'http://localhost:3000/', you will see the below page. With this, we have learned how to use the templating engine to control the page display.

Accessing MongoDB from Node.js

Second step in our lesson is 'Accessing MongoDB from Node'. For accessing MongoDB & the collections the node module 'Mongoose' will be used. Install the Mongoose and import it to the script 'mongocrud.js'. 

npm install mongoose
mongoose = require('mongoose');

Mongoose module provides the easy access to MongoDB. Opening the connection to the MongoDB is straight forward. For installing MongoDB visit www.mongodb.org. By default Mongo deamon will be running in the port 27017. MongoDB comes with a default database 'test'. Collections in MongoDB are similar to Tables in RDBMS and Mongo DB uses flexi schema approach. 

mongoose.connect('mongodb://localhost/test');

A MongoDB schema can be created like below using the Schema() from mongoose module. Create a collection containing the product details (id, name & price). Use the code below to create a reference to the 'product' object which uniquely represent a record in the collection. When this code executes, MongoDB creates a collection "Products'. Note: It adds 's' to the model name given. 

var productSchema = new mongoose.Schema({
	prdId: String,
	name: { type: String }, 
	price: Number
})

var Product = mongoose.model('Product', productSchema);
